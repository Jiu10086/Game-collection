# 🎮 Game Collection Tracker

เว็บแอปพลิเคชันสำหรับจัดการและติดตามคอลเลกชันเกมส่วนตัวของคุณ พัฒนาด้วย Flask, PostgreSQL และ Bootstrap 5

---

## ✨ ฟีเจอร์หลัก

- 🔐 **ระบบสมาชิก** — สมัคร, เข้าสู่ระบบ, ออกจากระบบ, เปลี่ยนรหัสผ่าน
- 📋 **จัดการเกม** — เพิ่ม, แก้ไข, ลบเกมในคอลเลกชันของคุณ
- 🔍 **ค้นหาแบบ Live** — พิมพ์ชื่อเกมแล้วผลลัพธ์ขึ้นทันทีโดยไม่ต้องรีเฟรชหน้า
- 📊 **Dashboard** — หน้าแรกแสดงสรุปสถิติเกมทั้งหมด (กำลังเล่น, เล่นจบแล้ว, อยากเล่น, เลิกเล่น)
- 🏷️ **เลือกได้หลายอย่าง** — เลือก Platform และ Genre ได้มากกว่า 1 อย่างต่อเกม
- ⭐ **ให้คะแนน** — ให้คะแนนเกมตั้งแต่ 1-10
- 📝 **โน้ต** — จดบันทึกส่วนตัวให้แต่ละเกมได้
- 👤 **โปรไฟล์** — แก้ไขข้อมูลส่วนตัวและรูปโปรไฟล์

---

## 🖥️ เทคโนโลยีที่ใช้

| เทคโนโลยี | หน้าที่ |
|---|---|
| **Flask** | เว็บเฟรมเวิร์กหลัก |
| **SQLAlchemy** | ORM สำหรับจัดการฐานข้อมูล |
| **PostgreSQL** | ฐานข้อมูล (โฮสต์บน Render.com) |
| **Flask-Login** | ระบบล็อกอิน/จัดการ session |
| **Flask-Bcrypt** | เข้ารหัสรหัสผ่าน |
| **Bootstrap 5** | ตกแต่งหน้าเว็บ |
| **Jinja2** | ระบบ template |

---

## 🧬 โครงสร้างโปรเจค

```
game/
├── main.py                  # จุดเริ่มต้นของแอป
├── app/
│   ├── __init__.py          # สร้าง Flask app (App Factory)
│   ├── extension.py         # ตั้งค่า db, login_manager, bcrypt
│   ├── models.py            # โมเดล User และ Game
│   ├── core/                # Blueprint หน้าแรก (Dashboard)
│   ├── games/               # Blueprint จัดการเกม (เพิ่ม/แก้/ลบ/ค้นหา/ดูรายละเอียด)
│   │   └── templates/games/
│   │       ├── index.html       # รายการเกมทั้งหมด
│   │       ├── new_game.html    # ฟอร์มเพิ่มเกม
│   │       ├── edit_game.html   # ฟอร์มแก้ไขเกม
│   │       ├── game_info.html   # หน้ารายละเอียดเกม
│   │       └── search.html      # ค้นหาเกม
│   ├── users/               # Blueprint ระบบสมาชิก (สมัคร/ล็อกอิน/โปรไฟล์)
│   ├── static/              # ไฟล์รูปภาพ
│   └── templates/           # Layout, Navbar และ Pagination
│       ├── layout.html          # โครงสร้างหลักของหน้าเว็บ
│       ├── navbar.html          # แถบนำทาง
│       └── pagination.html      # แถบเปลี่ยนหน้า (reusable)
├── .env                     # ตัวแปรสภาพแวดล้อม (DATABASE_URL, SECRET_KEY)
├── requirements.txt         # รายการ package ที่ต้องติดตั้ง
└── pyproject.toml           # ตั้งค่าโปรเจค
```

## 🌐 หน้าเว็บทั้งหมด

| เส้นทาง (Route) | คำอธิบาย |
|---|---|
| `/` | หน้าแรก — Dashboard แสดงสถิติและเกมล่าสุด |
| `/games` | รายการเกมทั้งหมดของคุณ |
| `/games/create` | เพิ่มเกมใหม่ |
| `/games/edit/<id>` | แก้ไขข้อมูลเกม |
| `/games/info/<id>` | หน้ารายละเอียดเกม |
| `/games/delete/<id>` | ลบเกม |
| `/games/search` | ค้นหาเกม (แบบ Live) |
| `/users/login` | เข้าสู่ระบบ |
| `/users/register` | สมัครสมาชิก |
| `/users/profile` | โปรไฟล์ผู้ใช้ |

## ℹ️ ข้อมูลที่เก็บในแต่ละเกม

| ฟิลด์ | คำอธิบาย |
|---|---|
| **Title** | ชื่อเกม |
| **Platform** | แพลตฟอร์ม — PC, PS5, PS4, Xbox, Nintendo Switch, Mobile (เลือกได้หลายอัน) |
| **Genre** | แนวเกม — RPG, Action, FPS, Puzzle, Adventure, Strategy, Sports, Indie, Soul-like, Horror และอื่นๆ (เลือกได้หลายอัน) |
| **Status** | สถานะ — กำลังเล่น, เล่นจบแล้ว, อยากเล่น, เลิกเล่น |
| **Rating** | คะแนน 1-10 |
| **Image URL** | ลิงก์รูปปกเกม |
| **Note** | บันทึกส่วนตัว |
